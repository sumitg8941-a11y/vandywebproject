# Product Overview

## Project Purpose
DealNamaa is a full-stack deals and coupons aggregation platform that connects users with retail offers across multiple geographic locations. The platform enables users to discover, browse, and redeem promotional offers from various retailers organized by country, state/region, and city.

## Value Proposition
- Centralized platform for discovering retail deals and promotional offers
- Geographic-based organization for location-specific deals
- Real-time tracking of offer engagement and user interactions
- Secure admin portal for content management
- Analytics and insights on offer performance

## Key Features

### User-Facing Features
- Geographic navigation (Country → State/Region → City → Retailers)
- Retailer browsing with category filtering
- Offer viewing with PDF flyers, coupon codes, and external links
- Global search across retailers and offers
- Offer feedback system (likes/dislikes)
- Tracked redirect links with UTM parameters
- Responsive Next.js frontend with server-side rendering

### Admin Features
- JWT-based authentication system (via admin.html interface)
- CRUD operations for all entities (Countries, States, Cities, Retailers, Offers)
- File upload system for offer materials (PDFs, images)
- Analytics dashboard with visit tracking
- Top performers tracking (retailers and offers by clicks)
- Feedback management system
- Web-based interface accessible at `/admin.html`

### Analytics & Tracking
- Site visit tracking
- Retailer click tracking
- Offer engagement metrics (clicks, views, time spent, pages viewed)
- User feedback collection (likes/dislikes)
- Outbound link tracking with UTM parameters

## Target Users

### End Users
- Bargain hunters looking for deals in their area
- Price-conscious shoppers seeking coupons and discounts
- Users browsing location-specific retail promotions

### Administrators
- Content managers maintaining retailer and offer databases
- Marketing teams analyzing offer performance
- Operations staff monitoring platform health and user engagement

## Use Cases
- Browse deals by geographic location
- Search for specific retailers or offer types
- View and download promotional flyers
- Copy coupon codes for online/offline use
- Track which offers are most popular
- Manage and update promotional content
- Analyze user engagement patterns
